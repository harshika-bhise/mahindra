# Mahindra

This is a Fullstack Mahindra Project. This application allows administrator to register, log in, and add vehical, edit it, update it etc. It aims to recreate the core functionalities of Mahindra.

Also users can apply to book service, test drive, etc.

And the most important, highlight of this project is user can compare between two vehicles.

## Tech Stack

**HTML, CSS, Bootstrap, JavaScript, jQuery, JSP, MySQL**
